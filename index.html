memlite

Persistent memory for AI agents.
A single static binary. Local embeddings, hybrid search, 16 MCP tools. Your AI's memory, everywhere.

Quick Start GitHub →
🧠

Hybrid Search

Full-text (FTS5) and vector embeddings fused with reciprocal rank fusion. Fast, relevant, and no external search service needed.

🔒

Local & Private

Everything runs on your machine. No API calls, no cloud, no telemetry. Your data never leaves your computer.

⚡

Single Binary

6.4 MB stripped. Statically links SQLite, sqlite-vec, llama.cpp, and md4c. Zero runtime dependencies. Built with Zig.

🏷️

Rich Tagging

JSON key/value tags on every memory. Filter search results by tags. Co-occurrence discovery with sibling tags.

📋

Soft-Delete & History

Every delete saves a snapshot. Full history table. Recover anything you've ever stored.

🔌

MCP Stdio

Native JSON-RPC over stdio. Works with Claude Desktop, Claude Code, Pi, and any MCP-compatible host.

Quick Start

# Download the latest release
curl -fsSL https://github.com/daemonicai/memlite/releases/latest

# Extract and install
tar -xzf memlite-*.tar.gz
install -m 0755 memlite-*/memlite ~/.local/bin/memlite

# Initialize (downloads model, creates database)
memlite init
See releases for pre-built binaries (macOS arm64, Linux x86_64, Linux arm64)

Use with Claude Code

Register memlite as a user-scoped MCP server — available across every Claude Code project on your machine.

claude mcp add memlite --scope user -- ~/.local/bin/memlite serve

Also works with Claude Desktop — see the README for config.

Use with Pi

The memlite-pi extension wraps memlite as native Pi tools with lifecycle management, guardrails for destructive operations, optional auto-save on session compaction, and ambient context injection from tagged memories. 

# Install memlite binary first, then:
pi install git:github.com/daemonicai/memlite-pi

Then restart Pi or run /reload. Tools appear in the tool list automatically.
Optional flags: --memlite-auto-save, --memlite-context, --memlite-unsafe, --memlite-path.
See github.com/daemonicai/memlite-pi for full docs.
